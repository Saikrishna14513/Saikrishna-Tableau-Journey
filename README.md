# Tableau Learning Journey

This repository documents my journey of learning Tableau for Data Analyst roles.  
It includes key concepts, practical examples, and mini projects.

---

##  Topics Covered

### 1. Connecting to Data
- Connecting to CSV, Excel, SQL, and other data sources  
- Understanding data types and table structures  

### 2. Filters & Sorting
- Apply filters to restrict data  
- Sort data alphabetically, numerically, or by custom order  

### 3. Charts
- Bar Chart, Line Chart, Pie Chart, Area Chart  
- Best practices for visualizing different types of data  

### 4. Dashboards & Stories
- Combine multiple charts into a dashboard  
- Use actions, filters, and interactive elements  
- Create storytelling dashboards  

### 5. Calculated Fields
- Create new fields using formulas  
- Conditional calculations using IF, CASE, etc.  

### 6. Parameters
- Create dynamic visuals using parameters  
- Apply parameters to filters, calculated fields, or reference lines  

---

## 🎯 Goal
To master Tableau and create interactive dashboards that communicate insights clearly.
